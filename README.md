Tycheck is a performance-optimization utility for Linux environments utilizing the apt package manager in order to ensure the speed and health of the operating system and its associated hardware are well-optimized. It makes use of varius tools to ensure that your system will be fast now and in the future.

### Installation and uninstallation
To install tycheck, simply run
```bash
sudo bash install.sh
```
To uninstall pass --uninstall (or -u)
